Por dónde empezar

Rutas según tu objetivo

Elegí el caso que más se parezca a tu situación actual.

01

Quiero cambiar de carrera o reconvertirme

  • Empezá con cursos cortos para validar el interés antes de comprometer tiempo.
  • Buscá tecnicaturas con salida laboral clara en tech, datos o administración.
  • Armá portfolio mientras estudiás: no esperes el título para demostrar criterio.
02

Quiero mejorar mi salida laboral rápido

  • Elegí cursos cortos con certificado y aplicación clara: Excel, administración, soporte, inglés laboral.
  • Armá un CV orientado a tareas reales y resultados, aunque tu experiencia sea chica.
  • Sumá un proyecto demostrable: una planilla, una web simple, un portfolio o un caso práctico.
03

Quiero un título oficial

  • Usá la Guía de Carreras Universitarias para validar institución, título y modalidad.
  • Revisá si la institución tiene SIED validado para modalidad a distancia.
  • Chequeá finales presenciales, tasas administrativas, becas y requisitos de ingreso.
04

No tengo plata para empezar

  • Priorizá plataformas gratuitas y autogestionadas: Punto Digital, Portal Empleo, IPFL.
  • Si una universidad cobra tasa administrativa, revisá becas o exenciones.
  • No pagues un curso caro hasta saber exactamente qué problema laboral querés resolver.

Antes de anotarte

Checklist antes de inscribirte

Esto evita perder tiempo, plata o motivación.

1
Verificá si es título oficial o certificado. No es lo mismo una licenciatura, una tecnicatura superior, un curso de extensión o un certificado de asistencia.
2
Confirmá modalidad real. "Virtual" no siempre significa 100% asincrónico. Puede haber finales presenciales, clases sincrónicas o encuentros obligatorios.
3
Revisá costos ocultos. Algunas instituciones no cobran arancel académico pero sí matrícula, tasa administrativa o uso de plataforma. También puede haber becas.
4
Miralo con objetivo laboral. Preguntate: ¿qué trabajo podría conseguir o mejorar con esto? ¿Qué proyecto puedo armar para demostrarlo?
5
Combiná formación + evidencia. Un certificado suma, pero un portfolio, una práctica, una planilla, una app o un caso real pueden demostrar criterio mejor que el papel solo.
